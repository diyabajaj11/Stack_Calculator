This is a C program that performs **expression conversion and evaluation** for infix, postfix, and prefix. The program includes options to:
- Convert expressions 
- Evaluate infix, postfix, and prefix expressions with single-digit numbers

- ## Features

Convert:
- Infix -> Postfix  
- Infix -> Prefix  
- Prefix -> Infix  
- Prefix -> Postfix  
- Postfix -> Infix  
- Postfix -> Prefix  

Evaluate:
- Infix Expression  
- Prefix Expression  
- Postfix Expression

Operators Description:
+ Addition
- Subtraction
* Multiplication
/ Division
^ Power
